(WIP) Cities Skylines Multiplayer Mod

This project is a large scope passion project currently done by only me and written in C#. Most rough designs of networking and game integration have been completed. To do most integration, a library called Harmony will be used to change function definitions after compilation by replacing the MSIL code.

The main blocker with this is creating an MSIL parsing library to find and replace specific statements. Next would be to sync game states by sending saved games over the network.
